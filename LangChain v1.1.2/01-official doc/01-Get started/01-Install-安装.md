# 安装 LangChain

## 基础安装

```bash
pip install -U langchain
# 需要 Python 3.10+
```

## 集成包安装

LangChain 提供了数百个 LLM 和数千个其他集成的集成。这些集成位于独立的提供程序包中。

```bash
# 安装 OpenAI 集成
pip install -U langchain-openai

# 安装 Anthropic 集成
pip install -U langchain-anthropic
```

## 下一步

现在您已经安装了 LangChain，可以按照快速入门指南开始使用。
