## 📋 生成规则

### **技术栈要求**
- Python 3.8+ 标准库
- 常用第三方库（requests, aiohttp, FastAPI, pytest等）
- 无需特殊 AI 接口，专注 Python 基础技能

### **输出规范**

- **输出目录**：`./Python/`
- **文档名格式**：`序号-说明-时间戳.ipynb`

### **代码要求**
1. **代码块独立**：每个代码块必须能独立运行，包含完整的导入和初始化
2. **标准库优先**：优先使用 Python 标准库，必要时使用第三方库
3. **完整示例**：从导入到验证的完整代码流程
4. **错误处理**：包含基础的异常处理机制
5. **实用参数**：使用实际可运行的参数和配置

### **笔记本结构要求**
每个笔记本包含以下部分：
1. **用途说明** - 明确学习目标和前置要求
2. **知识点覆盖** - 按照知识点清单的3部分结构
3. **独立代码块** - 每个代码块可单独运行
